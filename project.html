E-COMMERCE WEBSITE - SCHOOL PROJECT

In this project, I designed and developed a complete e-commerce website that allows users to browse products, add them to a shopping cart, proceed to checkout, and make purchases.

The website includes features such as user registration and authentication, product categories and filtering, product search functionality, order management, and secure payment integration.

I utilized PHP for server-side scripting to handle the dynamic generation of web pages and interact with the database. The database was designed to store product information, user details, and order data, ensuring seamless communication between the website and the backend. Overall, this project provided valuable hands-on experience in web development, database management, and the implementation of e-commerce functionalities.

Developpement PHP / HTML / CSS / JAVASCRIPT
